3. 总结

  • 父级使用iframe，需要使用@load绑定加载完毕要执行的函数
    <iframe src="./static/html/key-word.html" ref="iframe" @load="loaded" style="width: 100%;height: 100%" scrolling="no" frameborder="0" name="myFrame"></iframe>
  • 父页面获取iframe的内容
    iframe.contentWindow//获取iframe的window对象 iframe.contentWindow.document//获取iframe的document iframe.contentWindow.documentElement//获取iframe的html iframe.contentWindow.head//获取head iframe.contentWindow.body//获取body iframe.contentWindow.方法名//获取iframe子页面定义的全局方法
  • 子页面可以用vue,也可以不用，a，b两种方法如下：

  • a:父页面：
    let keywordFn=this.$refs.iframe.contentWindow.init; keywordFn(this.words);

  • a:子页面：
    function init(arr) {}

  • b:父页面：
    let keywordFn=this.$refs.iframe.contentWindow.vm.init; keywordFn(this.words);
  • b:子页面：
    window.vm = new Vue({
      name:’child’,
      methods:{
        init(){
        }
      }
    });
